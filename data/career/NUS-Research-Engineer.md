### 2.2 Vehicle Location and Size Tracking from Surveillance Camera

**Problem / Pain Points**
- Manual traffic monitoring was inefficient and error-prone  
- Need for accurate vehicle size and speed measurement from surveillance cameras  

**Actions**
- Implemented **YOLOv5-based wheel point extraction** using Python + OpenCV  
- Collected and analyzed 10,000+ vehicle data points via web crawling  
- Validated system on 5,000+ vehicles for width, length, and speed  

**Impact / Results**
- Achieved **>93% accuracy** for width and length  
- Achieved **>98% accuracy** for speed estimation  
- Enabled reliable traffic monitoring and dataset generation for research  

**Skills / Signals**
computer-vision, python, opencv, data-analysis, traffic-monitoring
